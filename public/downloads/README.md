# 다운로드 파일 폴더

이 폴더에 다운로드할 파일들을 넣어주세요.

## 지원하는 파일 형식
- PDF (.pdf)
- Word (.docx)
- Excel (.xlsx)
- PowerPoint (.pptx)
- 이미지 파일 (.jpg, .png, .gif)

## 파일명 규칙
- `friender_intro.pdf` - 프랜더 소개서
- `friender_portfolio.pdf` - 포트폴리오
- `friender_brochure.pdf` - 브로셔
- `friender_case_study.pdf` - 케이스 스터디

## 사용법
1. 파일을 이 폴더에 업로드
2. Contact.jsx에서 파일명을 확인
3. 웹사이트에서 다운로드 버튼 클릭

## 주의사항
- 파일 크기는 10MB 이하 권장
- 파일명에 공백이나 특수문자 사용 금지
- 한글 파일명 사용 가능
